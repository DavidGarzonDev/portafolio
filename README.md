## ⚙️ Instalación y Uso Local

1. 🔁 Clona este repositorio:
   ```bash
   git clone https://github.com/tu-usuario/mi-portafolio.git


2. 📂 Entra en la carpeta del proyecto:

   ```bash
   cd mi-portafolio
   ```

3. 📦 Instala las dependencias:

   ```bash
   npm install
   ```

4. 🚀 Ejecuta el proyecto:

   ```bash
   npm start
   ```
